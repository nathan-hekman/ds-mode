Alias for `/ds-mode`. Identical behavior.

- `/dsm` — activate at default (`full`)
- `/dsm lite|full|visual` — set mode
- `/dsm off` — disable for the session

See `commands/ds-mode.md` and `skills/ds-mode/SKILL.md`.
