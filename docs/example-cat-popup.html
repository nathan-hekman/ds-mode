Why cats knock things off tables

DS Mode · field guide

Hunt instinct

Paw → small object → it moves. Brain logs "prey behavior confirmed." Toy, pen, water glass: all read the same.

Tiny scientist

Cats run little gravity experiments. Edge of table is the lab bench. Cup is the variable. Result rarely disappoints.

Attention payout

You react every time. Yelling counts. Reaction = reward. Behavior gets reinforced even when you think you're punishing it.

Whisker check

Paws and whiskers probe edges, weight, texture. The cup falling is a side effect of testing where the table ends.

Bored cat, big stage

Indoor cat with no prey, no climb, no puzzle. Table becomes the stage. The fix isn't discipline — it's a taller cat tree, scheduled play, and food puzzles.
